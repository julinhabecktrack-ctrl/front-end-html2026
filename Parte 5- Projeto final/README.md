# Exercício 1 - Projeto Final (Parte 5)

## 🎯 Objetivo do Projeto
Este projeto foi desenvolvido como parte do aprendizado de desenvolvimento front-end com HTML5. O objetivo principal foi praticar a estruturação de páginas web, organização de diretórios, uso de links internos e externos, além da correta manipulação e exibição de imagens locais, garantindo boas práticas de acessibilidade.

## 📂 Estrutura de Pastas
A organização dos arquivos do repositório ficou estruturada da seguinte forma:

frontend/
└── Exercicio 1/
    └── Parte 5 - Projeto Final/
        ├── index.html
        ├── README.md
        └── img/
            └── sal.jpg

## 💻 Como Abrir o Projeto
Para visualizar o projeto localmente no seu computador, siga os passos abaixo:
1. Navegue até a pasta `Parte 5 - Projeto Final`.
2. Dê um duplo clique no arquivo `index.html` para abri-lo diretamente no seu navegador de preferência (Google Chrome, Firefox, Edge, etc.).
3. *Alternativa:* Caso utilize o VS Code, você pode clicar com o botão direito no `index.html` e selecionar a opção **"Open with Live Server"** para rodar um servidor local.

## 🔍 Validação e Revisão Feita
Durante o desenvolvimento, foram realizadas as seguintes verificações de qualidade:
* **Caminhos de Imagem:** Revisão do atributo `src` para garantir que a imagem `sal.jpg` está carregando corretamente a partir da subpasta `img/`.
* **Tags de Acessibilidade:** Inclusão do atributo `alt` descritivo na imagem para leitores de tela.
* **Links:** Teste manual de todos os links externos e dos links internos do tipo âncora para garantir que a navegação funciona sem quebrar.
* **Ortografia e Estilo:** Revisão gramatical cuidadosa de todos os textos e títulos apresentados na página.
